# 仿天猫购物网站

## 项目介绍

### 功能介绍

1. 用户注册登录功能
2. 商品分类显示功能
3. 商品查询显示功能
4. 购物车功能
5. 用户下单购买功能
6. 用户评价功能
7. 添加删除服装功能
8. 商品信息修改功能
9. 订单管理功能
10. 多卖家支持

### 运用知识

* SSM框架
* MySQL
* Tomcat
* 数据库基本查询
* 数据库连接池c3p0
* jstl标签库
* jsp,servlet
* javascript,jquery
* Maven
* git

### 搭建环境

JDK8.0 + MySQL + Tomcat

#### 使用方法

1. git clone `https://github.com/qxklove/tmall.git`
2. 使用IDEA导入打开
3. 导入sql文件tmall_ssm.sql
4. 部署Tomcat并启动
